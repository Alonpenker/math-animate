# Educational Storyboarding

Give each scene one instructional purpose and divide it into readable visual
phases. Each phase should establish, transform, compare, test, or conclude one
idea.

A storyboard phase does not need to map one-to-one to a subscene. A subscene
may develop one teaching idea through multiple related sequential actions on
the same visual setup. Use a later subscene for a separate comparison, reveal,
conclusion, layout, or teaching idea.

Use `show` when changing to a separate idea. Use `transform` when continuity
between complete visual snapshots teaches how the current visual becomes the
next visual. Use template-owned actions when motion or another sequential
change within the current visual setup teaches the idea directly.

For movement-based explanations, state what must remain fixed, what must move,
where it moves, and which labels or markers move with it. Prefer one dominant
change per transform or action sequence. Do not collapse equation change,
movement, comparison reveal, and conclusion into one snapshot transition when
they communicate distinct ideas.

End each scene with a stable snapshot whose main takeaway is understandable
without relying on narration or repeated explanatory text.
